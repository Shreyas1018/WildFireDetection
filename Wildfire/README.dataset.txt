This dataset is released by [AI for Mankind](https://aiformankind.com) in collaboration with [HPWREN](http://hpwren.ucsd.edu/) under a Creative Commons by Attribution Non-Commercial Share Alike license. The original dataset (and additional images without bounding boxes) can be found [in their GitHub repo](https://github.com/aiformankind/wildfire-smoke-dataset).

We have mirrored the dataset here for ease of download in a variety of [common computer vision formats](https://roboflow.com/formats).

To learn more about this dataset and its possible applications in fighting wildfires, see this [case study of Abhishek Ghosh's wildfire detection model](https://blog.roboflow.com/fighting-wildfires).
